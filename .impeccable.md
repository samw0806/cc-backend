## Design Context

### Users
有一定技术背景的运营人员会使用这个界面连接 Claude Agent Server，查看历史会话、继续对话、理解当前模型执行到了哪一步，以及在需要时处理工具权限请求。

### Brand Personality
克制、专业、可靠。界面应让人感觉像一款稳定的工作工具，而不是强调娱乐感或炫技的 AI 演示页。

### Aesthetic Direction
参考 ChatGPT 的对话界面信息架构和清爽度，但保持更偏内部工具的秩序感。仅提供浅色主题，以白底为主，使用克制的灰阶和少量蓝绿色状态色，不使用夸张渐变、霓虹、毛玻璃或强装饰。

### Design Principles
1. 对话优先：主区首先服务于阅读和连续交流，状态信息不应压过消息本身。
2. 状态透明：思考中、工具调用、工具结果、权限请求都应清晰可见，且能被快速扫描。
3. 专业克制：视觉层次明确，但避免夸张色彩、过重阴影和炫目动效。
4. 低学习成本：默认行为接近熟悉的聊天产品，复杂连接配置收纳在设置抽屉中。
5. 连续性优先：会话历史、恢复和当前连接状态必须稳定，刷新后应尽量少丢上下文。
